# EVA Foundation – Technical Design & ConOps
## Canonical Source Summary (Extracted)

This document defines the technical design and Concept of Operations (ConOps)
for ESDC’s Virtual Assistant (EVA) Foundation, hosted in Azure Cloud within
ESDC’s secure infrastructure.

EVA Foundation consists of two core applications:
- EVA Chat
- EVA Domain Assistant (EVA DA)

The platform is designed to enhance employee productivity using Generative AI
while complying with Government of Canada (GoC) standards for:
- Security
- Privacy
- Accessibility
- Bilingualism

EVA is approved by the Enterprise Architecture Review Board (EARB)
(January 29, 2025) and is hosted in a Protected B Azure environment.

Key architectural principles:
- Accessibility-first
- Secure-by-design
- No external public exposure
- Vendor-agnostic cloud capability
- Human-in-the-loop accountability

This document is evergreen and will evolve as EVA capabilities mature.
