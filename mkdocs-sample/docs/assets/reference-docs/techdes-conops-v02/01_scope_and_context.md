# EVA Foundation – Scope and Context

## Business Context
EVA is led by the AI Centre of Enablement (AICoE) within IITB,
in collaboration with Shared Services Canada (SSC).

It supports employees across:
- ESDC
- Labour
- Service Canada

The platform addresses security and compliance risks associated
with public AI tools by providing a secure internal alternative.

---

## Expected Outcomes

### For Employees
- Increased productivity
- Improved decision-making (human validation required)
- Increased GenAI literacy
- Accessible and inclusive tooling

### For Clients
- Faster service delivery
- Improved accuracy
- Better client interactions
- 24/7 availability (via employee enablement)

### For IITB / Department
- Responsible AI integration
- Cost recovery via usage-based billing
- Scalability and future-readiness
- Strengthened data security and compliance
